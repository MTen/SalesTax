UserInput takes initial input and checks it for glaring errors (null, non-string entry) if passes creates UserInput OBJECT
	
OrderParser ABSTRACT_OBJECT_OR_METHOD takes each line from UserInput OBJECT and puts it into an UserInput.inputArray ( .split(\n))

InputMatcher evaluates each index of inputArray, creates a Transaction object for each 
	
CALCULATOR_OBJECT takes each Transaction object modifies the price based on keywords and puts it the Batch object 
	
Batch Object is printed out for user.
